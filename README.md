# Product Management App

Product Management App is a web application developed with Node.js and Express, designed for managing products through a simple and user-friendly interface.

The application allows users to register, log in, and manage product records. Authenticated users can add new products, view the product list, edit existing products, and delete products. Product data is stored in a MySQL database, while user authentication is handled using Passport.js, express-session, and bcrypt for password hashing.

The project follows a basic MVC-style structure, using routes for request handling, models for database operations, EJS templates for rendering pages, and Bootstrap/CSS for styling the interface.

## Main Features

* User registration and login
* Session-based authentication
* Password hashing with bcrypt
* Product management: add, view, edit and delete products
* MySQL database connection
* Server-side rendering with EJS
* Responsive interface using Bootstrap
* Custom CSS styling

## Technologies Used

* Node.js
* Express.js
* MySQL
* EJS
* Passport.js
* bcrypt
* express-session
* Bootstrap
* HTML / CSS
